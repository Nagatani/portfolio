## 自分用メモ

### Install

#### 1. Install Gridsome CLI tool if you don't have

`npm install --global @gridsome/cli`

#### 2. Install this starter

1. `gridsome develop` to start local dev server at `http://localhost:8080`
2. `gridsome build` to build
3. Happy coding 🎉🙌
